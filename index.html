OMORI Dotfiles

WELCOME TO WHITE SPACE.

This is a collection of OMORI-themed dotfiles for Arch Linux, featuring configurations for:

  • Hyprland (Window Manager)
  • Alacritty (Terminal)
  • Rofi (Application Launcher)
  • Waybar (Status Bar)
  • Spicetify (Spotify Theme)
  • And more...

Installation

Run these commands in your terminal:

curl -LO https://buttermiilk.github.io/omori-dotfiles/setup.sh
chmod +x ./setup.sh
./setup.sh

Note: This script is designed to work only on Arch Linux.

Requirements

This setup requires a clean Arch Linux installation. The script will install all necessary dependencies.
